对语文水平仅仅只有89分的我来说，组织语言写出来确实是个很大的挑战，可是呢，为了装X和学习，那我就想挑战一下自己了。

Markdown 记录

Markdown 标题

​ Markdown使用#来表示标题，共有六级，#的数量表示代表的级数。

​ 但是我看到菜鸟教程上有用“=”的形式来设置一级标题，用“-”的形式来设置二级标题，但是我自己试了试，不知道为什么没标题效果。

Markdown 段落

​ 段落没有特殊格式，直接编写即可，可以使用空行来换行，段落开头可以用一个tab来就行段首设置。

字体

​ 斜体字 ：左右各用一个*号或者_包裹

​ 粗体字 ：左右各用两个*号或者_包裹

​ 粗斜体字 ：左右各用三个*号或者_包裹

分割线

​ 在一行中用三个或者三个以上的*或者-或者_

删除线

​ 删除线 ：左右各用两个~包裹

下划线

​ 下划线 ：用类似html的 <u> 标签包裹即可

脚注

​ 脚注： [ ^文本内容] 像这样写出来之后，鼠标指向会有一个好像网址的东西，点击就会弹出一行东西，填写就行啦

Markdown 列表

  • 无序列表标记：可以用 *或+或-后边加空格就可以使用
  1. 有序列表标记：用数字. 后面加空格在写文本的方式使用

Markdown 区块

> 区块用>后加空格使用，而且可以嵌套

Markdown 代码

代码 代码片段可以用单`包裹

代码块 可以用```来包裹

Markdown 链接

链接 两种形式

  1. [链接名称](链接地址)

  2. [链接名称]

    [链接名称]:链接地址

Markdown 图片

  1. ![alt 属性文本](图片地址)

  2. 图片作为链接 [属性文本][网址或者变量]

    若为变量[变量]：图片地址

Markdown 表格

typora制作表格的时候只需要用|把表头隔开

例如：|表头|表头|
